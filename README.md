# SpringBoot-게시판
## SpringBoot를 이용한 게시판 만들기

## 📖설명
- 기존에 실습했던 사원 관리 프로젝트에서 MVC 패턴과 CRUD 기능을 한번 더 손에 익히기 위해 회원가입기능과 게시판기능을 추가하는 프로젝트를 기획하게 되었습니다
